# Linux AutoCleaner v1.0

**A safe, one-click optimization script for Linux beginners.**

## Features
- Clean package caches (supports Arch, Debian, Ubuntu, Fedora, openSUSE)
- Remove old logs
- Clear thumbnail cache
- Remove orphan packages (Arch only)
- Show system info (uptime + disk usage)
- Beginner-friendly
